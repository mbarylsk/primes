# primes

Collection of libraries and tools to experiment with various aspects of primes and other numbers.

* primes.py - library with key prime related methods
* calculations.py - library with key calculations
* unittests.py - unit tests for primes and calculations
* t_prime_numbers.txt - list of prime numbers
* t_nonprime_numbers.txt - list of composite numbers
* /projects - folders with specific projects
